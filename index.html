KPMP Applications

Overview:

The applications described here were developed by the Data Visualization Center (DVC) portion of the KPMP project. We have developed a wide array of applications in order to support the gathering, dissemination and display of the data generated through this project. These pages will contain information about each of the projects as well as documentation for any developers who wish to make use of our works.
Applications:
KPMP Data Lake Uploader

Used by the entire consortium to store and share all of the raw and processed data that is being generated.
My Data

Web portal for KPMP partcipants to view their biopsy results as whole slide images.
Digital Pathology Repository (DPR)

Web portal for browsing KPMP biopsy whole slide images, organized by case and slide properties.
Data Manager Dashboard (DMD)

Application for KPMP data managers to review and data between the Data Lake and the Knowledge Environment / Atlas.
