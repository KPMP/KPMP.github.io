Repositories

List of the repositories required to run this application.
Repository Description
hubble-web The GUI layer for the Atlas Spatial Viewer
pegasus-data The service layer for the Atlas Spatial Viewer
heavens-docker This repository holds all of the Docker containers and docker-compose scripts for running our applications
knowledge-environment (private repository) Contains schema DTDs for the data stores used in the Atlas Repository, Explorer, and Spatial Viewer

Data Stores

S3
We store our ome-tiffs, which are generated from tiffs in our processing pipeline, in an S3 bucket. This bucket title is "kpmp-knowledge-environment" and is structured using package IDs.
